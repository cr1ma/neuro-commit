## Про проєкт

NeuroCommit — це мінімалістичний веб-інструмент, який допомагає розробникам створювати якісні повідомлення для комітів. Просто вставте вивід `git diff`, і отримайте структурований промт для вашої улюбленої нейромережі.

## Особливості

- **🎯 Простота** — Чистий інтерфейс без зайвих елементів
- **⚡ Швидкість** — Миттєва генерація промпту
- **🎨 Підсвічування** — Візуальне відображення змін у коді
- **📋 Одним кліком** — Копіювання промпту та команд
- **🌓 Теми** — Підтримка світлої та темної теми
- **💾 Автозбереження** — Ваші зміни зберігаються локально

## Швидкий старт

### Базове використання

```bash
# Подивіться ваші зміни
git diff --staged

# Скопіюйте вивід і вставте у NeuroCommit
```

### Розширене використання (PowerShell)

```powershell
# Виключення lock-файлів з diff
[Console]::OutputEncoding = [System.Text.UTF8Encoding]::new($false)
git --no-pager diff --staged --color=never -- . ':(glob,exclude)**/package-lock.json' | Out-File -FilePath .\diff.patch -Encoding utf8
```

## Робочий процес

1. **Зробіть зміни** у вашому коді
2. **Додайте файли** до stage: `git add .`
3. **Отримайте diff**: `git diff --staged`
4. **Вставте у NeuroCommit** та отримайте промт
5. **Скопіюйте промт** та використайте з AI
6. **Створіть коміт** з згенерованим повідомленням
